## Projeyi Çalıştırma
#### Frontend 
````
git clone https://github.com/ferhatergun/Google-text-to-speech.git
cd client
npm install
npm run dev
````
#### Backend
````
cd server
npm install
npm start
````

### .env dosyası
````
CLOUD_API_KEY = "YOUR_GOOGLE_API_KEY"
````
